# 🧠 Brain Tumor Detection System (Machine Learning + Android)

## 📌 Overview

This project is an **Automatic Brain Tumor Detection System** that uses **Machine Learning and Deep Learning techniques** to classify MRI brain images and detect the presence of tumors.

It also includes an **Android application** built in Java that allows users to easily upload and analyze MRI scans on mobile devices.

---

## 🚀 Key Features

### 🤖 Machine Learning Model

* Developed using **Python, Keras, and TensorFlow**
* Trained on MRI image dataset for binary classification (Tumor / No Tumor)
* Achieved **97% → 98.1% accuracy** after optimization and fine-tuning
* Improved performance through:

  * Hyperparameter tuning
  * Data augmentation
  * Model architecture improvements

### 📱 Android Application

* Built using **Java & XML (Android Studio)**
* Simple and user-friendly UI for medical image input
* Displays prediction results instantly
* Integrated with backend ML model for real-time inference

### 🔗 System Integration

* Seamless connection between ML model and Android app
* Efficient data flow for image processing and prediction
* Optimized for usability and responsiveness
